# 📊 Student Score Prediction using Linear Regression

## 🚀 Project Overview
This project predicts a student's exam score based on the number of hours they studied using **Linear Regression**.

It is a beginner-friendly Machine Learning project that demonstrates:
- Data preprocessing
- Model training
- Prediction
- Visualization

---

## 📂 Dataset
- Input Feature: **Hours Studied**
- Target Variable: **Exam Score**

---

## ⚙️ Technologies Used
- Python 🐍
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

---

## 🧠 Machine Learning Model
- Algorithm: **Linear Regression**
- Purpose: Predict continuous values (Exam Score)

---

## 🔄 Project Workflow

1. Import Libraries  
2. Load Dataset  
3. Data Preprocessing  
4. Train-Test Split  
5. Model Training  
6. Prediction  
7. Visualization  
   - Regression Line  
   - Actual vs Predicted Graph  

---

## 📈 Visualization

### 🔹 Regression Line
- Blue dots → Actual data
- Red line → Best-fit prediction line

### 🔹 Actual vs Predicted
- Shows how close predictions are to real values
- Points near diagonal → Good model performance

---

## 🎯 Results
- Model successfully predicts exam scores based on study hours
- Visualization helps understand model accuracy

---

## 📌 Key Learnings
- Basics of Linear Regression
- Model training and prediction
- Data visualization using matplotlib
- Evaluating model performance visually
